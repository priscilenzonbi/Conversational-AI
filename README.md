# Conversational AI

A hands-on journey building conversational AI systems, growing in complexity over time.

## Projects

- **01-basic-chatbot** — A chatbot built with Streamlit and the Gemini API, with conversation memory and a system instruction for controlled behavior.
- **02-rag-chatbot** — A retrieval-augmented chatbot that answers from a fixed knowledge base using embeddings and cosine similarity, and refuses to answer beyond its sources.
- **03-document-chat** — Upload your own PDF or text file and chat with it. The app chunks and embeds the document on the fly, retrieves relevant passages, and answers only from what you uploaded, with a relevance threshold so off-topic questions get a clean "I don't know."

## Concepts covered so far

Tokenization, generation controls, conversation memory, system instructions, hosted model APIs, embeddings, semantic retrieval, chunking, and grounded answering.

Built as part of my path into conversational AI and AI safety.
